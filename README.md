# LingoCraft

## Features

- **Interactive Globe Map** This globe allows the user to spin, zoom in, and play with it. The globe shows the pins of the languages. 
- **Historical and Statistical facts**: Each pin-point will be featured by the name of the language, language family and the countries and regions where the language is spoken. Moreover, the details features the short description about the language, alphabets, endangerment status and so on.
- **Villager Bot**: Users can ask some related and interesting questions.
- **Admin Feature**: Admin feature is included so that the admins can update the information on the website.
- **Responsive Design**: A basic responsive navbar and optimized interfaces for mobile and desktop viewing.

## Prerequisites

Before you begin, ensure you have the following installed:
- Python 3.8 or later
- pip (Python package installer)

## Getting Started

### 1. Clone the repository

git clone https://github.com/IDevanshi/Calgary-Hacks-2026.git

### 2. Install dependencies

pip install -r requirements.txt
npm install

#### 3. Environment Variables
Create a .env file in the root directory of the project and add the following environment variables:

SECRET_KEY=add_your_secret_key_here

### 4. Run the application

1) Go to the frontendTest directory on your terminal.
2) Type: npm install
3) Type: npm run dev
4) This will start the application on localhost by default and you may explore the application
